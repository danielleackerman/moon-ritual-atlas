# Moon Ritual Atlas

A mobile-friendly ritual atlas for lunar ritual experiences.

Current experience:

- Full Moon in Capricorn Ritual Index
- Dawn / Dusk modes
- Ritual Path Finder
- Ritual index cards
- Guided notecard ritual pages
- Swipe navigation
- Copy / export / receipt flow

## Deploy

This is currently a static HTML site. Deploy with Vercel by importing this GitHub repository, or open `index.html` locally in a browser.
